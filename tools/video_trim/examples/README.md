# video-cortar examples

Default 18-minute cut:

```bat
..\video-cortar.bat video.mp4
```

Custom duration:

```bat
..\video-cortar.bat video.mp4 --duration 00:05:00
```

Input and output paths with spaces:

```bat
..\video-cortar.bat "C:\Videos\mi video.mp4" --output "C:\Videos\mi video_18min.mp4"
```
