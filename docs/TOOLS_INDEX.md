# GenSkill Tools Index

| Tool | Utility | Example |
|---|---|---|
| `text-count.bat` | Counts words, characters, and lines from `.txt` files or direct text. | `text-count.bat archivo.txt` |
| `img-web.bat` | Optimizes local images for web as compressed WebP or JPG copies with safe resizing. | `img-web.bat foto.jpg` |
| `video-2web.bat` | Optimizes local videos for web as H.264/AAC `.mp4` copies using FFmpeg. | `video-2web.bat video.mp4` |
| `video-cortar.bat` | Cuts local videos to a duration using FFmpeg without re-encoding. | `video-cortar.bat video.mp4` |
